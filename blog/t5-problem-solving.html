Problem-Solving

Tell your non-tech friend a story about a time you got blocked on a simple problem:

This has happened a number of times to me. I can’t remember the specifics, but there was one time where I couldn’t get my code to work and it was because of a fairly straightforward syntax error. I didn’t realise that loading the page I was working on in my browser and going into the console would tell me what line of my Javascript file the error was on, and I ended up asking for help from one of my peers and having them find this for me by using the console. I found this process quite frustrating, as I didn’t really know what problem-solving techniques to use other than asking for help from others. It would be good to get more of an idea of standard techniques for figuring out what’s going on when code doesn’t run properly (e.g. loading the page and using the console, which is one thing I learned from this).
